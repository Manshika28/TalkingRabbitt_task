# Talking Rabbitt MVP

Conversational Intelligence Layer for Business Data

## Problem
Business leaders struggle to extract insights quickly from dashboards like PowerBI and Tableau.

## Solution
Talking Rabbitt enables leaders to ask natural language questions directly on business data.

## Features
- Upload CSV dataset
- Ask questions in natural language
- Instant insights
- Automated visualizations

## Tech Stack
- Streamlit
- Pandas
- PandasAI
- OpenAI API
- Matplotlib

## How to Run

pip install -r requirements.txt
streamlit run app.py
